# Air Quality Dashboard
## How to use
### 1.Setup Environment - Shell/Terminal
```
cd git clone https://github.com/ShineyVamp/final-project.git
pipenv install
pipenv shell
pip install -r requirements.txt
```
### 2.Buka dashboard
```
cd submission/dashboard/
streamlit run dashboard.py
```
