# Air Quality Dashboard
## How to use
### 1.Clone repository
```
cd git clone https://github.com/ShineyVamp/final-project.git
cd final-project
pip install -r requirements.txt
```
### or Download file
```
cd final-project-main
pipenv install
pipenv shell
pip install -r requirements.txt
```
### 2.Open dashboard
```
cd dashboard
streamlit run dashboard.py
```
